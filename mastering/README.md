# マスタリングカウンター

アルバム内の意図的な曲間差を保ちながら、全曲へ同じ共通ゲインと仕上げ設定を適用するブラウザ完結ツール。

## 実装状態

- Gate 0〜2実装済み
- 音声処理はWeb Workerで一曲ずつ実行
- 出力はPCM WAV 24-bit / 48 kHz
- 複数曲はJSZip 3.10.1でZIP化
- 推定出力500 MB超は個別ダウンロードへ切り替え
- 音声ファイルを送信するネットワークAPIは使用しない

## ローカル確認

リポジトリ直下で静的サーバを起動し、`/mastering/`を開く。

```powershell
python -m http.server 8765
```

自動テスト：

```powershell
npm test
```

## 既知の制限

- 参考LUFSはアルバム内比較用で、BS.1770準拠の測定値ではない。
- Peak Safeはサンプルピーク基準で、検証済みTrue Peakではない。
- 読み込み可能な圧縮形式はブラウザのデコーダーに依存する。
- 大容量処理は端末メモリに依存する。
- Edge／Firefox／Safariなど、実際に使うブラウザでの追加QAは必要に応じて行う。
